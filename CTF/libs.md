Если случается так, что при gcc сборке ругается на отсутсвие библиотек. Нужно:

1. Смотрим, какой библиотеки нет, например, lreadline.
2. Ставим их на свой хост. Их название должно быть типа libreadline-dev.
3. Затем при компиляции передаем линки в конце команды:

```
gcc -std=c11 -o aes *.c -lreadline -lsqlite3
```

4. Как можно заметить, тут еще определяется версия C. С этим вопросм лучше потом идти к AI.
